## ✅ Code Convention

### Getting Started

1. `clone` the repository,

```
$ git clone "https://github.com/HeeYeonKim98/wanted_pre_onboarding.git"
```

2. `Install` dependencies,

```
$ npm install
```

3. VScode prettier

```
$ touch .prettierrc
```

4. `start` the project,

```
$ npm start
```
